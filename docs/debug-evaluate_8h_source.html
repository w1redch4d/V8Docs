61 // This class builds a context chain for evaluation of expressions
62 // in debugger.
63 // The scope chain leading up to a breakpoint where evaluation occurs
64 // looks like:
65 // - [a mix of with, catch and block scopes]
66 // - [function stack + context]
67 // - [outer context]
68 // The builder materializes all stack variables into properties of objects;
69 // the expression is then evaluated as if it is inside a series of 'with'
70 // statements using those objects. To this end, the builder builds a new
71 // context chain, based on a scope chain:
72 // - every With and Catch scope begets a cloned context
73 // - Block scope begets one or two contexts:
74 // - if a block has context-allocated varaibles, its context is cloned
75 // - stack locals are materizalized as a With context
76 // - Local scope begets a With context for materizalized locals, chained to
77 // original function context. Original function context is the end of
78 // the chain.
