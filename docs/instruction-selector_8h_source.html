588 // Used in pattern matching during code generation.
589 // Check if {node} can be covered while generating code for the current
590 // instruction. A node can be covered if the {user} of the node has the only
591 // edge, the two are in the same basic block, and there are no side-effects
592 // in-between. The last check is crucial for soundness.
593 // For pure nodes, CanCover(a,b) is checked to avoid duplicated execution:
594 // If this is not the case, code for b must still be generated for other
595 // users, and fusing is unlikely to improve performance.
596 bool CanCover(node_t user, node_t node) const;
597
598 bool CanCoverProtectedLoad(node_t user, node_t node) const;

600 // Used in pattern matching during code generation.
601 // This function checks that {node} and {user} are in the same basic block,
602 // and that {user} is the only user of {node} in this basic block. This
603 // check guarantees that there are no users of {node} scheduled between
604 // {node} and {user}, and thus we can select a single instruction for both
605 // nodes, if such an instruction exists. This check can be used for example
606 // when selecting instructions for:
607 // n = Int32Add(a, b)
608 // c = Word32Compare(n, 0, cond)
609 // Branch(c, true_label, false_label)
610 // Here we can generate a flag-setting add instruction, even if the add has
611 // uses in other basic blocks, since the flag-setting add instruction will
612 // still generate the result of the addition and not just set the flags.
613 // However, if we had uses of the add in the same basic block, we could have:
614 // n = Int32Add(a, b)
615 // o = OtherOp(n, ...)
616 // c = Word32Compare(n, 0, cond)
617 // Branch(c, true_label, false_label)
618 // where we cannot select the add and the compare together. If we were to
619 // select a flag-setting add instruction for Word32Compare and Int32Add while
620 // visiting Word32Compare, we would then have to select an instruction for
621 // OtherOp *afterwards*, which means we would attempt to use the result of
622 // the add before we have defined it.
623 bool IsOnlyUserOfNodeInSameBlock(node_t user, node_t node) const;
