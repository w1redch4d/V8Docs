v8
V8 is Google’s open source high-performance JavaScript and WebAssembly engine, written in C++. It is used in Chrome and in Node.js, among others. It implements ECMAScript and WebAssembly, and runs on Windows, macOS, and Linux systems that use x64, IA-32, or ARM processors. V8 can be embedded into any C++ application.

252 new_disposable_stack->set_stack(disposable_stack->stack());
253 new_disposable_stack->set_length(disposable_stack->length());
254 new_disposable_stack->set_state(DisposableStackState::kPending);

260 disposable_stack->set_stack(ReadOnlyRoots(isolate).empty_fixed_array());
